## 📖 About the Project

This is a **production-ready SaaS template** featuring:

- Secure JWT authentication with refresh tokens  
- Modular full-stack architecture  
- API request queueing and error handling  
- Stripe-ready billing integration  
- DevOps-ready deployment structure  

Perfect for building scalable SaaS applications or showcasing a **hireable portfolio project**.

---

## ⚡ Features

- ✅ Authentication (JWT + refresh tokens + Google OAuth)  
- ✅ Role-based access (admin/user)  
- ✅ Fully typed API with Axios helper functions  
- ✅ File uploads via Cloudinary  
- ✅ Payments via Stripe  
- ✅ Real-time data caching with React Query  
- ✅ State management with Redux Toolkit  
- ✅ TailwindCSS & Framer Motion animations  
- ✅ Logging & monitoring (Winston + error handling)
