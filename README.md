## Featured Project

### [CartLy — Production-Grade eCommerce Platform](https://mcartly.vercel.app/)

A full-stack MERN eCommerce platform built to production standards — multi-role access control, real-time payments, and enterprise-level security architecture.

| Area | Implementation |
|---|---|
| **Roles** | 5 user roles: Buyer · Seller · Admin · Superadmin · Warehouse |
| **Authentication** | JWT access + refresh tokens with Redis blacklisting · Google OAuth 2.0 |
| **Security** | Helmet · rate limiting · XSS · HPP · NoSQL injection protection |
| **Payments** | Stripe PaymentIntents with webhook validation |
| **Infrastructure** | Docker + Nginx reverse proxy · Cloudinary media storage · Email notifications |
| **Warehouse** | Parcel scanner with full check-in audit trail |
